If you don't like using websites for something that can be put into a command line interface application you're invited to install the command line interface of tophone. To install you must have the Go programming language installed. After installation run:

go get github.com/mini-eggs/tophone/cmd/tophone

This will build a binary and place it at $HOME/go/bin/tophone. If you've installed Go correctly this will already be in your path. If not I recommend taking a trip to DuckDuckGo. Run tophone help for a list of options.

The easiest way to send text to your mobile device from your computer.

Start by creating an account (or signing into your existing account) then creating a "note" - these notes are immediately sent to the number you've provided.
